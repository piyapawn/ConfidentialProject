CASE 03

search engine เช่น google มีอำนาจในการ
เปลี่ยนมุมมองของเราต่อโลกอย่างไร นำโดย
Robert Epstein ได้จัดหาคน 2150 คนที่ยังไม่ได้

search engine ที่ทำขึ้นมาพิเศษที่เรียกว่า
Kadoodle เพื่อให้ข้อมูลเกี่ยวกับผู้ลงสมัคร
ก่อนตัดสินใจ โดยทำให้ Kadoodle มีความ
ไม่เป็นกลาง และผู้เข้าทำการทดลองไม่ทราบ

ผู้เข้าร่วมใช้เวลาส่วนใหญ่ของพวกเขาอ่านเว็บไซต์

ให้ความสนใจกับ link ที่อยู่ในรายการล่างๆ

kadoodle ลำเอียง

CASE 04

การใช้ algorithm เพื่อวิเคราะห์ว่า

ลูกค้าคนไหนกำลังตั้งครรภ์ของ target

การใช้ algorithm เพื่อวิเคราะห์ว่าลูกค้าคนไหน
กำลังตั้งครรภ์ของ target ในปี 2002
target จะใช้ algorithm ให้คะแนนว่าลูกค้าผู้หญิง
คนไหนที่มีแนวโน้มจะตั้งครรภ์ผ่านเว็บไซต์ของ
พวกเขา หากคะแนนความเป็นไปได้ผ่านเกณฑ์
ร้านค้าจะส่งคูปองโดยอัตโนมัติไปให้ผู้หญิงคนนั้น
ซึ่งเป็นคูปองสินค้าที่เกี่ยวข้องกับการตั้งครรภ์

ผลปรากฎว่ามีกรณีที่ target ทราบว่าลูกสาวของ
ครอบครัวหนึ่งกำลังตั้งครรภ์ก่อนพ่อแม่ของเธอ
เสียอีก ซึ่งในทีแรกพ่อของเด็กหญิงคนนั้นได้มา
ร้องเรียนว่าทำไม target ถึงส่งคูปองสิ่งของ
เกี่ยวกับการตั้งครรภ์ให้ลูกของเขา ด้วยความกังวล
ว่าเป็นการสนับสนุนการท้องก่อนวัยอันควร
หรือเปล่า และเขาก็ได้กลับมาขอโทษบริษัท
เมื่อพบว่าลูกของเขาตั้งครรภ์จริงๆ
